Pooch Madness
=============
A fun and fundraising app for our furry friends.
Written in node by the Waltham MA JS meetup group

##Setup instructions:

###Install Node:
```
brew install node
```

###Install Bower:
```
npm install -g bower
```

###Install Dependancies:
```
npm install
```

###Install Bootstrap:
```
bower install bootstrap
```

###Start Server:
```
node app.js
```

##Optional

###Install Supervisor
```
npm install supervisor -g
```

###Start Server with Supervisor:
```
supervisor app.js
```
